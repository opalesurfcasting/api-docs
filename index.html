GFW API Docs

API Docs for GFW services

Global Fishing Watch processes positional AIS messages from vessels around the world, applying state-of-the-art machine learning and probabilistic algorithms to classify and extract information from the huge amount of data world-wide vessels generate. We expose this information through various standard JSON RESTful services.

This site contains a service map and interactive API documentation for the RESTfull services exposed by Global Fishing Watch. Not all of these services are open to the public, get in touch with us if you want to use any of them!

Our environments

Development

The development environment is a very unstable environment where we publish unpolished data and unfinished features. This environment is protected, so you will only be able to access these services if you work at GFW.
Pelagos: Vessel fishing effort map tiling service Events: Vessel event aggregation query service

Staging

The staging environment is a much more stable environment where we publish data and features for our QA and data analysis teams to work on. It's also where certain features are made available for beta testing to our partners.
Pelagos: Vessel fishing effort map tiling service

Production

The production environment where we publish data and features the whole world to work with. Is the most stable environment, closely monitored and updated through a controlled release cycle.
Pelagos: Vessel fishing effort map tiling service
